Important Questions – Computer Architecture

Module 1 – Indian Contribution & Basics

  • Explain Pingala’s contribution in binary system.
  • Panini’s grammar and its relation to rule-based systems.
  • Modern Indian contributors in computer architecture.

Module 2 – Digital Electronics

  • Number system conversions (Binary, Octal, Hexadecimal).
  • Binary arithmetic and BCD addition.
  • ASCII, EBCDIC, Gray code.
  • Boolean algebra and logic gates.
  • Combinational and sequential circuits.

Module 3 – Combinational & Sequential Circuits

  • Half Adder and Full Adder.
  • Encoders and Decoders.
  • Multiplexer and Demultiplexer.
  • Flip-flops (SR, JK, D, T).
  • Shift registers and Counters.

Module 4 – Basic Computer Organization

  • Instruction cycles and addressing modes.
  • Machine language vs Assembly language.
  • Register Transfer Language.
  • Arithmetic and Logic micro-operations.

Module 5 – Processor & Control Unit

  • Hardwired vs Microprogrammed Control Unit.
  • RISC vs CISC architecture.
  • Instruction formats and stack organization.
  • Concept of pipelining and hazards.

Module 6 – Memory & I/O Systems

  • Memory hierarchy and cache memory.
  • Virtual memory and associative memory.
  • DMA transfer and interrupts.
  • Processor vs memory speed.
  • Memory management techniques.
