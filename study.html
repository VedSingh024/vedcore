Operating System Important Questions

Unit 1 – Introduction to Operating System

  • Define Operating System and explain its functions.
  • Explain types of Operating Systems with examples.
  • Difference between Batch, Multiprogramming and Time Sharing OS.
  • Explain Resource Management in OS.
  • History and evolution of Operating Systems.

Unit 2 – Process Management

  • What is a Process? Explain Process States.
  • Explain Process Control Block (PCB).
  • Compare Preemptive and Non-Preemptive Scheduling.
  • Explain FCFS, SJF, Round Robin scheduling algorithms.
  • Define Deadlock and explain its necessary conditions.
  • Methods for Deadlock Prevention and Avoidance.

Unit 3 – Memory Management

  • Explain Logical and Physical Address Space.
  • What is Paging? Explain with diagram.
  • Difference between Paging and Segmentation.
  • Explain Virtual Memory and Demand Paging.
  • Page Replacement Algorithms (FIFO, LRU, Optimal).
  • What is Fragmentation? Types of Fragmentation.

Unit 4 – File & Disk Management

  • Explain File System structure.
  • Types of File Allocation Methods.
  • Directory Structures in OS.
  • Disk Scheduling Algorithms (FCFS, SCAN, C-SCAN).
  • Explain Linux file system structure.
  • Difference between Windows and Linux.
